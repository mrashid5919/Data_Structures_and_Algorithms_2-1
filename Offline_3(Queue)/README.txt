The purpose of the source files:

QueueADT.java - An interface which contains the public methods used by Aqueue and Lqueue classes

Aqueue.java - Array based implementation of QueueADT

Lqueue.java -  Linked list based implementation of QueueADT

Main.java - Demo main method showcasing the use of Aqueue and Lqueue classes

Bank.java - Implementing bank scenario using my QueueADT
