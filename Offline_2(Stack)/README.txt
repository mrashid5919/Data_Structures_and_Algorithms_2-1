The purpose of the source files:

StackADT.java - An interface which contains the public methods used by Arr and LL classes

Arr.java - Array based implementation of StackADT

LL.java -  Linked list based implementation of StackADT

Main.java - Demo main method showcasing the use of Arr and LL classes

Dw.java - Implementing dishwasher operations using my StackADT
